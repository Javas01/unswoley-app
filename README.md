# unswoley-app

### Requirements

- `node` >=v14
- `yarn`

### Running

```bash
$ cd unswoley-app
$ yarn
$ yarn ios
```

### Recommended Tools

- NVM
- VS Code
  - plugins: `ES Lint`, `Prettier`, `GitLens`
  - workspace: _settings.json_
    ```
    {
      "editor.defaultFormatter": "esbenp.prettier-vscode",
      "editor.formatOnSave": true,
      "editor.codeActionsOnSave": {
        "source.fixAll": true
      }
    }
    ```
